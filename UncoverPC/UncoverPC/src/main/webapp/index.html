    About

    Functionality

    UncoverPC allows the user to save money when they buy a new computer. Building computers is often way cheaper than buying a prebuilt, which is why our software will automatically find similar components for you to build for much cheaper without sacraficing performance

    Usage

    To use UncoverPC, click "Get Started" button, which will forward you to the search page. Then in the search bar , input a link to a prebuilt computer from BestBuy. The search process may take awhile.
